# 📦 Inventário Industrial

Sistema web desenvolvido para realizar inventários industriais de forma rápida, segura e integrada ao Supabase.

O projeto foi criado para eliminar planilhas manuais e permitir que a contagem seja realizada diretamente no navegador, utilizando leitores de código de barras ou digitação manual.

---

## 🚀 Tecnologias

- React
- TypeScript
- Supabase
- Vercel

---

## ✨ Funcionalidades

- ✅ Contagem cega
- ✅ Leitura por código de barras
- ✅ Digitação manual
- ✅ Importação de estoque
- ✅ Exportação em CSV
- ✅ Histórico das contagens
- ✅ Dashboard
- ✅ Controle por usuário
